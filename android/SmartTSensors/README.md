SmartT Android
================

Overview
---------

The android application is reading from the couchdb server which serves a 
JSONARRAY which include:
 - SensorID
 - Power in voltages
 - Temperature in Celsuis
 - type of sensor(weather/power)
 - Light in LUX
 - Carbon Monixide
 - Rain
 - Humidity
